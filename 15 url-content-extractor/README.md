# URL Content Extractor

This is a simple Streamlit application that extracts text content from a URL.

## Features
- Enter a URL
- Fetch webpage content
- Display the first 200 characters of the text
- Display the total word count
- Show an error message if the URL fetch fails

## How to Run

Install dependencies:
pip install streamlit requests beautifulsoup4

Run the app:
streamlit run app.py
